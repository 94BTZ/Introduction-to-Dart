void main() {

}

### Variables

#### String
```Dart
var name = 'My Name';
```

#### Integer
```Dart
var year = 1998;
```

#### Double
```Dart

```

```Dart
var myList = ['Item1', 'Item2', 'Item3', 'Item4'];
```
